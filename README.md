# wdio-docx-evid

wdio-docx-evid is a Javascript library to generate test evidence documents

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install.

```bash
npm i wdio-docx-evid
```

## Usage

```javascript

browser.takeScreenshot();
```
